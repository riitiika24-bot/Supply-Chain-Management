# Supply Chain Management System

**Author:** DEEPSHIKHA 
**Project Type:** DBMS / Oracle SQL

## Description
The Supply Chain Management System handles inventory movement, supplier orders, logistics, and delivery tracking across multiple locations. 
It is designed to manage stock levels, supplier relationships, order fulfillment, and shipment tracking using Oracle SQL as the database backend.

## Objectives
- Track inventory across multiple warehouses and locations.
- Manage supplier information and purchase orders.
- Monitor shipments, logistics, and delivery status.
- Generate reports for stock levels, order fulfillment, and supplier performance.

## Database Design (Tables)
- Warehouses
- Suppliers
- Products
- Inventory
- Purchase_Orders
- Order_Items
- Shipments
- Delivery_Tracking

## How to Run
1. Install Oracle Database or use Oracle Live SQL.
2. Run `tables.sql` to create tables.
3. Run `insert_data.sql` to add sample data.
4. Run `queries.sql` to run example reports and checks.
